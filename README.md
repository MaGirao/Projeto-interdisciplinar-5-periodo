Plataforma digital desenvolvida como Projeto Interdisciplinar do curso de Análise e Desenvolvimento de Sistemas, com o objetivo de apoiar o bem-estar emocional de jovens por meio de conteúdos informativos, autoavaliação emocional e agendamento psicossocial.
 Acesse a aplicação:

https://magirao.github.io/Projeto-interdisciplinar-5-periodo/

Funcionalidades

* Cadastro de usuários
* Login
* Conteúdos sobre saúde mental
* Autoavaliação emocional
* Agendamento psicossocial
* Armazenamento local de dados
  
Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* LocalStorage
* Git
* GitHub Pages

Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/MaGirao/Projeto-interdisciplinar-5-periodo.git
```

2. Acesse a pasta do projeto:

```bash
cd Projeto-interdisciplinar-5-periodo
```

3. Abra o arquivo `index.html` em seu navegador.

Observações

Este projeto foi desenvolvido como um MVP (Minimum Viable Product) para fins acadêmicos.

Os dados são armazenados localmente utilizando LocalStorage e não são enviados para servidores externos.
